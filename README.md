# Brewery Search

Search your favorite breweries!

## Installation

```bash
npm install
```

```bash
npm start
```

## Usage

```python
import foobar

foobar.pluralize('word') # returns 'words'
foobar.pluralize('goose') # returns 'geese'
foobar.singularize('phenomena') # returns 'phenomenon'
```

## Built With

[Open Brewery API](https://www.openbrewerydb.org/)
