# S.O.S. (Search Out Suds)

Search your favorite breweries!

## Installation

```bash
npm install
```

```bash
npm start
```

## Built With

[Open Brewery API](https://www.openbrewerydb.org/)

- React
- Express
- SCSS

## Future Features

- Pagination
- Results per page options
- A seperate list of favorited breweries
